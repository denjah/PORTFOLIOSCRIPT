# Changelog

Все важные изменения в проекте документируются в этом файле.

## [1.2.0] - 02.01.2026

### Изменено
- **Логика работы фона**: Фон теперь скроллится вместе с контентом (убрано fixed позиционирование)
- **Чередование фонов**: Поддержка двух версий фона (А и Б) с автоматическим чередованием
- **Динамическая подгрузка**: Фон автоматически подгружается по мере скролла до конца контента
- CSS: Убрано `body::before` с fixed позиционированием, добавлен `.background-container` со скроллируемыми сегментами
- JavaScript: Полностью переработана функция `loadBackground()` с поддержкой чередования А → Б → А → Б...

### Технические детали
- Фон состоит из динамически создаваемых сегментов
- Каждый сегмент имеет высоту оригинального изображения
- MutationObserver отслеживает изменения контента для автоматической подгрузки фона
- Fallback: если версия Б отсутствует, используется только версия А
- Оптимизация производительности через `requestAnimationFrame` и throttling

## [1.1.0] - 02.01.2026

### Добавлено
- **Переключение формата карточек**: Теперь поддерживаются два формата - 3:4 (portrait) и 1:1 (square)
- Панель управления форматом в правом верхнем углу интерфейса
- CSS классы `.card--portrait` и `.card--square` для управления пропорциями
- Функция `handleFormatChange(format)` для программного переключения формата
- Кнопки переключения форматов с визуальной индикацией активного формата
- Обновлена документация с описанием работы с форматами

### Изменено
- `CONFIG.cardFormat` теперь определяет формат карточек по умолчанию
- Функция `createCardElement()` автоматически применяет нужный класс формата
- Обновлены консольные подсказки с информацией о форматах
- Расширены советы по использованию разных форматов

### Технические детали
- Формат применяется ко всем карточкам одновременно
- Переключение происходит мгновенно без перезагрузки страницы
- Панель управления скрывается при печати/экспорте
- Состояние формата сохраняется в CONFIG и может быть экспортировано

## [1.0.0] - 02.01.2026

### Добавлено
- Базовая HTML структура с 4 разделами
- CSS Grid система (4 колонки)
- Система фоновых изображений с зеркальным дублированием
- Автоматическая загрузка и распределение карточек по группам
- Drag & Drop для перестановки карточек
- Кнопки управления: "первая карточка" и "swap режим"
- Декоративные иконки (70×70px)
- Dev Mode для отладки сетки
- Консольные команды и горячие клавиши
- Полная документация (README.md)
- Структура папок для ресурсов

### Функционал
- 4 раздела: Вступление, Процесс, Галерея, Подвал
- Редактируемые текстовые блоки (contenteditable)
- Первая карточка группы занимает 2 колонки
- Автоматический рендеринг подвала (4 первые карточки)
- Экспорт в PDF через window.print()
